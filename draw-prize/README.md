# 抽奖系统（可自定义版）

这是一个可配置的抽奖页面原型，用于实现自定义抽奖主题、奖品配置以及指定中奖名单的能力。

## 功能目标

- 支持创建多个抽奖主题
- 支持自定义奖品列表（名称、数量、概率或固定分配）
- 支持导入参与人员名单
- 支持手动指定中奖结果（用于活动控制）
- 支持随机抽取模式
