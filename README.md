# Jin10 Plus

Static prototype workspace for the Jin10 Plus membership product.

## Structure

- `pages/`: all preview HTML pages
- `assets/images/`: image assets
- `assets/styles/`: CSS files
- `assets/scripts/`: static data scripts
- `docs/`: product and planning docs

## Main entry points

- `index.html`
- `pages/plus-prototype.html`
- `pages/plus-library.html`
- `pages/plus-topics.html`
- `pages/plus-market-desk.html`
- `pages/jin10-main-plus-feed.html`

## Rules

- User-facing pages in this workspace should only render final copy.
- Internal notes, requirement explanations, and prototype remarks should stay out of the UI.
